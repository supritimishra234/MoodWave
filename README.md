MoodWave
  
MoodWave is a simple frontend project made using HTML, CSS and JavaScript.  
The idea of this project is to give music and wallpapers based on the user's mood.  
When the user selects a mood, the website opens a Spotify playlist and also shows related wallpapers which can be downloaded.

Features

- Mood based music playlists
- Mood based wallpapers
- Download wallpaper option
- Dark / Light mode toggle
- Search bar in navbar
- Simple UI

Tech used- HTML,CSS,JavaScript
